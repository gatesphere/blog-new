pila - a stack based language

July 13, 2012
tags: pila, language, io, stack, forth

Hello everyone,

Just thought I'd let you know about my new pet language project: a stack based language inspired by FORTH and Lisp. I'm calling it pila, which is a controversial name in Portuguese, but means "pile" or "stack" in Italian.

If you're at all interested, you can check it out on github.
